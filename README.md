## Getting Started

### Prerequisites
Make sure you have:
- [Node.js](https://nodejs.org/) (vXX or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/your-repo-name.git

# Go into the project folder
cd your-repo-name

# Install dependencies
npm install

# Start the development server
npm run dev
```
